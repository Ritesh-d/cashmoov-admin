File Name Format :

VOUCHER_{{fileNameFormat}}-DD-MM-YYYY .txt

Where :

DD-MM-YYYY :

Date of Upload

Data Format :

From Serial Number|To Serial Number|Quantity

Where :

From Serial Number - Mandatory

To Serial Number - Optional

Quantity - Mandatory

Note :

  1. File name should be unique.
  2. Voucher from and to serial number should be of the same product as mentioned in the record
  3. Total number of voucher in a record should be same as total voucher.
